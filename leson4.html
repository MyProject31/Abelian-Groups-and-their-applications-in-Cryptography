Discrete Logarithm Problem

Definition

The discrete logarithm problem is a fundamental hard problem in cryptography. It is defined in a finite cyclic group G. The goal is to find an integer x such that:

g^x = y

where y ∈ G and g is a generator of G.

It is denoted by:

x = log_g(y)

We say that x is the discrete logarithm of y to the base g.

Example: Find x such that 2^x ≡ 4 mod 7

Diffie Hellman Key Exchange

Definition

The Diffie Hellman key exchange is a protocol in cryptography that lets two people share a secret key securely, even if their communication is not secure.

Steps of the Diffie Hellman Key Exchange Protocol

  1. Alice and Bob agree on public parameters: a large prime number p and a generator g.
  2. Alice chooses a private key a, and Bob chooses a private key b.
  3. Alice computes: x = g^a mod p Bob computes: y = g^b mod p
  4. Alice and Bob exchange their public values x and y.
  5. Alice receives y, and Bob receives x.
  6. Alice computes: K_A = y^a mod p Bob computes: K_B = x^b mod p

As a result, Alice and Bob obtain the same shared secret key:

K_A = K_B

This key is used for encryption.

ElGamal Public-Key Encryption

Definition

ElGamal is a public key encryption method based on the discrete logarithm problem. It uses a group G and a generator g.

Steps of the ElGamal Encryption

  1. Alice chooses a large prime number p and a generator g.
  2. Alice chooses a private key a.
  3. Alice computes: y = g^a mod p Public key: (p, g, y)
    Private key: a
  4. Bob wants to send a message m. He chooses a random number k.
  5. Bob computes: c1 = g^k mod p c2 = m · y^k mod p
  6. Bob sends the ciphertext (c1, c2) to Alice.
  7. Alice decrypts: m = c2 · (c1^a)^(-1) mod p

RSA Public-Key Encryption

RSA is a public key cryptosystem. It is named after its inventors Rivest, Shamir, and Adleman.

Steps of RSA Public Key Encryption

  • Alice asks Bob to create a public key and a private key.
  • Bob chooses two prime numbers p and q.
    n = p × q
    φ(n) = (p − 1)(q − 1)
  • Bob chooses e such that: 1 < e < φ(n) and gcd(e, φ(n)) = 1
  • Bob computes: d × e ≡ 1 mod φ(n)
  • Public key: (n, e)
    Private key: d
  • Alice encrypts: c = m^e mod n
  • Bob decrypts: m = c^d mod n
