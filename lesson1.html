Definition

A binary operation on a non-empty set E is an application that combines two elements of E and provides another element of the same set.

Examples

  1. The usual addition on the set of integers ℤ.
  2. The usual multiplication on the set of real numbers ℝ.
  3. Subtraction is not a binary operation on the set of natural numbers ℕ.

Properties of a Binary Operation ⋆

  1. The law ⋆ is associative if, for all a, b, c ∈ E: (a ⋆ b) ⋆ c = a ⋆ (b ⋆ c).
  2. An element e ∈ E is the identity element of ⋆ if for any a ∈ E: e ⋆ a = a ⋆ e = a.
  3. An element a ∈ E is invertible if there exists a⁻¹ ∈ E such that: a ⋆ a⁻¹ = a⁻¹ ⋆ a = e, where e is the identity element.
  4. The law ⋆ is commutative if for all a, b ∈ E: a ⋆ b = b ⋆ a.

Groups and Abelian Groups

Definition

Let E be a non-empty set with a composition law ⋆. The pair (E, ⋆) is a group if and only if:

  1. The operation ⋆ is an internal composition law on E (a ⋆ b ∈ E for all a, b ∈ E).
  2. The operation ⋆ is associative: (a ⋆ b) ⋆ c = a ⋆ (b ⋆ c)for all a, b ,c ∈ E.
  3. e ⋆ a = a ⋆ e = a.
  4. Every element a ∈ E has an inverse a⁻¹ ∈ E: a ⋆ a⁻¹ = a⁻¹ ⋆ a = e.

Abelian Group

If a ⋆ b = b ⋆ a for every pair (a, b) ∈ E, the group is called an Abelian group (commutative group).

Examples

  • ℝ, ℤ, ℚ, ℂ are groups with respect to the usual addition.
  • ℝ*, ℚ*, ℂ* are groups with respect to the usual multiplication.
  • ℕ is not a group under addition because not every element has an additive inverse in ℕ.

Proposition

Let E be a non-empty set together with a composition law *. We say that E is a group under the operation * if the following properties are satisfied:

  1. ICL (Closure): It means a * b ∈ E for all a, b ∈ E.
  2. Associativity: It means (a * b) * c = a * (b * c) for all a, b, c ∈ E.
  3. Identity element: It means there exists e ∈ E such that a * e = a for all a ∈ E.
  4. Inverse element: It means for all a ∈ E, there exists a⁻¹> ∈ E such that a * a⁻¹ = e, where e is the identity element.

Remark

This proposition is valid only if the identity element and the inverse element are found on the same side. For example, if we find the right identity element, we should find the right inverse element, not the left.

Proof

We want to prove that if a * e = a and a * a⁻¹ = e, then e * a = a and a⁻¹* a = e.

Let a ∈ E. Then there exists a⁻¹ ∈ E such that

a * a⁻¹ = e

where e is the identity element. Since a⁻¹ ∈ E, there exists b ∈ E such that

a⁻¹ * b = e

Now we prove that e * a = a. Using associativity, we have:

e * a = e * (a * e) = e * (a * a⁻¹* b) = e * (e * b) = (e * e) * b = e * b = (a * a⁻¹) * b = a * (a⁻¹ * b) = a * e = a

Thus, e * a = a.

Next, we show that a⁻¹ * a = e. We have:

a⁻¹ * a = a⁻¹ * a * e = a⁻¹ * a * a⁻¹ * b = a⁻¹ * e * b = a⁻¹* b = e

Definition

Let E be a non-empty set together with a binary operation * on E.

  • If the operation * is associative (and closed on E), then the pair (E,*) is called a semigroup.
  • If a semigroup admits an identity element, then it is called a monoid.

Group Morphism

Definition

Let (E, ⋆) and (S, *) be two groups. An application f : E → S is called a group morphism if, for all a,b ∈ E, we have

f(a ⋆ b) = f(a) * f(b)

Definition

Let f be a group morphism from the group E to the group S. Then:

  1. If E = S and (⋆ = *), then we say that f is an endomorphism.
  2. If f is bijective, then we say that f is an isomorphism.
  3. If f is a bijective endomorphism, then we say that f is an automorphism.

Properties

Let f : (G, *) → (H, ⋆) be a group morphism. Then the following properties are satisfied:

  1. f(e_E) = e_S, where e_E and e_S are respectively the identity elements of E and S.
  2. For every a ∈ E, we have f(a⁻¹) = (f(a))⁻¹.
  3. If f is an isomorphism, then its inverse function f⁻¹ is also an isomorphism from (S, *) to (E, ⋆).
  4. If H is a subgroup of E, then f(H) is a subgroup of S.
  5. If T is a subgroup of S, then f⁻¹(T) is a subgroup of E.

Subgroups

Definition

Let E be a group under the operation ⋆, and S be a non-empty subset of E. We say that S is a subgroup of E if it is a group itself under the operation of E.

Proposition

A subset S is a subgroup of E if and only if:

  1. The set S is non-empty (at least contains the identity element of E).
  2. For all pairs (a,b) ∈ S, we have a ⋆ b ∈ S.
  3. For all a ∈ S, its inverse a⁻¹ ∈ S.

Proposition

A subset S is a subgroup of E if and only if:

  1. The set S is non-empty (at least contains the identity element of E).
  2. For all pairs (a,b) ∈ S, we have a ⋆ b⁻¹ ∈ S.

Remark

  • The intersection of subgroups of (E, ⋆) is also a subgroup of (E, ⋆).
  • The union of subgroups of a group (E, ⋆) is not necessarily a subgroup of (E, ⋆).

Finite groups and finite abelian groups

Definition (order of a group)

Let (E, ⋆) be a group. Its order is the number of elements of E and it is denoted by |E|.

Remark

The order of a group E can be finite or infinite; in the last case we say that E has infinite order.

Examples

  1. The set of integers ℤ has infinite order because its number of elements is infinite.
  2. The set S = {0,1,2,3,4,5} is finite, and its order is simply the number of its elements which is 6.

Definition (order of an element)

The order of an element a in a group E is the smallest positive integer n such that aⁿ = e, where e is the identity element of E.

Remark

We say that an element a has infinite order if there is no positive integer n such that aⁿ = e.

Definition

  1. The group (E, ⋆) is said to be a finite group if it has a finite number of elements (i.e., finite order).
  2. The group (E, ⋆) is said to be a finite abelian group if it is finite and the commutative property is satisfied.
  3. In a finite group, every element has finite order, whereas in an infinite group, some elements can have finite order.

Cyclic groups

Definition

A group E is called a cyclic group if there exists an element a ∈ E such that every element of E can be written as a power of a. The element a is called a generator of E.

  • If the group E is a multiplicative cyclic group, then E = ⟨a⟩ = {aⁿ | n ∈ ℤ}.
  • If the group E is an additive cyclic group, then E = ⟨a⟩ = {n a | n ∈ ℤ}.

Examples

  1. The group ℤ₆ = {0,1,2,3,4,5} under addition modulo 6 is cyclic and its generators are 1 or 5.
  2. The group E = {1, i, -1, -i} under multiplication is a cyclic group of order 4 with generator i.

Remark

An integer k is a generator of the additive group ℤₙ if and only if gcd(n,k) = 1.

Properties

  1. A cyclic group can have more than one generator.
  2. Every cyclic group is an abelian group.
  3. Every subgroup of a cyclic group is a cyclic group.
  4. In a finite cyclic group, the order of an element divides the order of the group.

Theorem (Lagrange's Theorem)

Let E be a finite group and S be a subgroup of E. Then the order of S divides the order of E, that is:

|S| | |E|

Basic Number Theory and Arithmetic

Integers and Divisibility

Definition

Let a and b be two integers (elements of ℤ). We say that a divides b, and we write a | b, if there exists an integer k such that:

b = a × k

Examples

  • 3 | 12 because there exists k = 4 such that 12 = 3 × 4.
  • 5 ∤ 12 because there is no integer k such that 12 = 5 × k.

Properties of Divisibility

  • If a | b and a | c, then a | (b + c) and a | (b - c).
  • If a | b, then a | (b × c) for any integer c.
  • If a | b and b | c, then a | c.
  • If a | b and b | a, then a = b or a = -b.

Prime and Composite Numbers

Definition

An integer n > 1 is called a prime number if it has only 1 and itself as divisors. Otherwise, it is a composite number.

Remark

The number 1 is neither prime nor composite.

Examples

  • Prime numbers: 2, 3, 5, 7, 11, 13.
  • Composite numbers: 4, 6, 8, 9, 10, 12.

Prime Factorization

If a number is composite, it can be factored into numbers greater than 1. These factors can be prime or composite. A factorization into primes is called prime factorization.

Theorem (Fundamental Theorem of Arithmetic)

Every integer a > 1 is either a prime number or can be written uniquely as a product of primes:

a = p₁^k₁ × p₂^k₂ × … × pₙ^kₙ

where p₁, p₂, …, pₙ are primes and k₁, k₂, …, kₙ are positive integers.

Example

Let a = 60:

60 = 2 × 2 × 3 × 5 = 2² × 3 × 5

Check if the number is prime or not and calculate the prime factors.

Greatest Common Divisor (GCD)

Let a and b be integers, not both zero. The greatest common divisor of a and b, denoted gcd(a,b), is the greatest positive integer dividing both.

Least Common Multiple (LCM)

Let a and b be non-zero integers. The least common multiple, denoted lcm(a,b), is the smallest positive integer divisible by both.

Examples

  • GCD example: a = 12, b = 18 → gcd(12,18) = 6
  • LCM example: a = 12, b = 18 → lcm(12,18) = 36

Euclidean Algorithm

For integers a > b > 0, divide a by b to get remainder r:

a = bq + r, 0 ≤ r < b

Then gcd(a,b) = gcd(b,r). Repeat until r = 0. The last non-zero remainder is the GCD.

Example

Find gcd(48,18):

48 = 18 × 2 + 12
18 = 12 × 1 + 6
12 = 6 × 2 + 0

So, gcd(48,18) = 6.

Coprime (Relatively Prime) Integers

Two integers a and b are coprime if gcd(a,b) = 1.

Examples

  • 8 and 15 are coprime because gcd(8,15) = 1.
  • 12 and 18 are not coprime because gcd(12,18) = 6 ≠ 1.

Bézout's Theorem

For integers a and b, there exist u and v such that:

a·u + b·v = gcd(a,b)

If a and b are coprime, then:

a·u + b·v = 1

Extended Euclidean Algorithm

This finds integers u and v such that a·u + b·v = gcd(a,b).

Example

Find u and v for 252u + 105v = gcd(252,105):

252 = 105 × 2 + 42
105 = 42 × 2 + 21
42 = 21 × 2 + 0

So gcd(252,105) = 21. Back-substituting:

21 = 105 × 5 - 252 × 2

Thus, u = -2, v = 5 → 252(-2) + 105(5) = 21
