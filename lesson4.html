Discrete Logarithm Problem

Definition

The discrete logarithm problem is a fundamental hard problem in cryptography. It is defined in a finite cyclic group G. The goal is to find an integer x such that:

g^x = y

where y ∈ G and g is a generator of G.

It is denoted by:

x = log_g(y)

We say that x is the discrete logarithm of y to the base g.

Example: Find x such that 2^x ≡ 4 mod 7

Diffie Hellman Key Exchange

Definition

The Diffie Hellman key exchange is a protocol in cryptography that lets two people share a secret key securely, even if their communication is not secure.

Steps of the Diffie Hellman Key Exchange Protocol

  1. Alice and Bob agree on public parameters: a large prime number p and a generator g.
  2. Alice chooses a private key a, and Bob chooses a private key b.
  3. Alice computes: x = g^a mod p Bob computes: y = g^b mod p
  4. Alice and Bob exchange their public values x and y.
  5. Alice computes: K_A = y^a mod p Bob computes: K_B = x^b mod p

Shared secret:

K_A = K_B

ElGamal Public-Key Encryption

Definition

ElGamal is a public key encryption method based on the discrete logarithm problem.

Steps of the ElGamal Encryption

  1. Alice chooses a prime p and generator g.
  2. Alice chooses private key a.
  3. y = g^a mod p Public key: (p, g, y)
  4. Bob encrypts message m using random k.
  5. c1 = g^k mod p c2 = m · y^k mod p
  6. Alice decrypts: m = c2 · (c1^a)^(-1) mod p

RSA Public-Key Encryption

RSA is a public key cryptosystem.

Steps of RSA Public Key Encryption

  • Choose primes p and q.
  • n = p × q, φ(n) = (p − 1)(q − 1)
  • Choose e such that gcd(e, φ(n)) = 1
  • Compute d × e ≡ 1 mod φ(n)
  • Public key: (n, e), Private key: d
  • Encrypt: c = m^e mod n
  • Decrypt: m = c^d mod n

RSA Python Implementation

import math

def is_prime(x):
    if x < 2:
        return False
    for i in range(2, int(math.sqrt(x)) + 1):
        if x % i == 0:
            return False
    return True

def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a % b)

def mod_inverse(e, phi):
    d = 1
    while (d * e) % phi != 1:
        d += 1
    return d

p = int(input("Enter a prime number p: "))
q = int(input("Enter a prime number q: "))

if not is_prime(p) or not is_prime(q):
    print("Error: p and q must be prime numbers!")
    exit()

n = p * q
phi = (p - 1) * (q - 1)

e = 3
while gcd(e, phi) != 1:
    e += 2

d = mod_inverse(e, phi)

print("Public key:", (e, n))
print("Private key:", (d, n))

message = input("Enter message: ")
encrypted = []

for ch in message:
    encrypted.append((ord(ch) ** e) % n)

print("Encrypted message:", encrypted)

decrypted = ""

for num in encrypted:
    decrypted += chr((num ** d) % n)

print("Decrypted message:", decrypted)
